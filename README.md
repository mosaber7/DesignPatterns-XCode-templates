## General info
You will be able to add the files and dependicies for the Arch you want by just choosing it, and it will be attached to your XCode so you can them in any projects you want

## How To Use

### 1- Go to your Finder
### 2- Hit `cmd` + `shift` + `G`
### 3- Paste the following path `/Users/[ YOUR USERNAME ]/Library/Developer/Xcode`
